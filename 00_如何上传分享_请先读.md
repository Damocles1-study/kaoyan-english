# 英语一复盘中心 · 上传与分享说明

> 本文件夹已是一个 **git 仓库** + 一套静态 HTML 网站。`index.html` 是入口，其余每个 html 是一篇复盘页。
> 本质就是网页，所以可以跨设备、可以分享。三种方式按需选。

## ⚠️ 先看：版权与隐私

这些页面含**真题原文全文 + 全文翻译**。
- **自己用 / 发给个别研友**：没问题。
- **公开发布到网上**（Public 仓库）：理论上有真题版权风险。要发布就用**私有仓库**（你的 GitHub 学生包有 Pro，可从私有仓库开 Pages），或只点对点分享。
- 页面不含你的姓名/学校，这点无隐私顾虑。

---

## 方式 A：只想自己多设备看（最省事，零配置）

把整个「阅读复盘」文件夹放进 **iCloud 云盘 / 坚果云 / 百度网盘同步盘**，
手机、iPad、另一台电脑上用浏览器打开 `index.html` 即可。改动自动同步。

## 方式 B：想要一个网址，随时随地打开、也能发人（GitHub Pages）

前提：GitHub 账号（你有，还有学生包）。想公开就跳过"私有"设置；想稳妥就建私有仓库。

```bash
# 1. 在本文件夹打开终端（路径已经是 git 仓库，不用再 init）
cd "/Users/zyx/Downloads/新威考研-资料/11408相关/英语一/阅读复盘"

# 2. 在 GitHub 网页新建一个仓库，名字建议用英文：kaoyan-english
#    （公开=Public / 私有=Private，私有需 Pro 才能开 Pages）

# 3. 关联并推送（把 <你的用户名> 换成自己的）
git remote add origin https://github.com/<你的用户名>/kaoyan-english.git
git branch -M main
git push -u origin main
```

4. 仓库 **Settings → Pages** → Source 选 `main` 分支、根目录 `/` → Save。
5. 等 1–2 分钟，访问 `https://<你的用户名>.github.io/kaoyan-english/` —— 上线完成。
   把这个网址发给研友即可（私有仓库的 Pages 需对方也有权限，纯自用更适合私有）。

**以后每加一篇复盘页**（Claude 生成后）：
```bash
git add -A && git commit -m "add: 2007 T1" && git push
```
一两分钟后网址自动更新。

## 方式 C：临时发某一篇给别人看

单个 html 是自包含的（样式脚本全在文件里），直接把某个 `.html` 通过微信/QQ 发过去，
对方用浏览器打开就能用——但**词卡、判题都正常，唯一失效的是"返回复盘中心"链接**（因为没有同目录的 index）。
